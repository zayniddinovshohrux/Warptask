# 🌀 WarpTask AI - Sog'lom Hayot va Vazifa Yordamchisi

## 🌍 3 til: O'zbekcha, Русский, English

**WarpTask AI** - bu sizning shaxsiy sog'lom hayot, vazifalarni rejalashtirish va kundalik motivatsiya yordamchingiz. 100% offline ishlaydi, mahfiylikni qo'llab-quvvatlaydi.

## ✨ Xususiyatlari

- 🤖 **Local AI** - Microsoft Phi-2 modeli, internet talab qilmaydi
- 🌐 **3 til** - O'zbek, Rus, Ingliz tillarida ishlash
- 💬 **Smart Chat** - Tabiiy suhbat, kontekstni eslash
- 📋 **Vazifa Rejalashtirish** - Murakkab vazifalarni qadamlarga ajratish
- 🍎 **Sog'lom Hayot** - Ovqatlanish, mashq, uyqu maslahatlari
- ⚡ **Tezkor** - CPU da ham tez ishlaydi
- 🔒 **Xavfsiz** - Ma'lumotlar lokal saqlanadi

## 🚀 O'rnatish

```bash
# 1. Repository ni klonlash
git clone https://github.com/sizning/warp-task-ai.git
cd warp-task-ai

# 2. Virtual environment
python -m venv venv

# 3. Faollashtirish
# Windows: venv\Scripts\activate
# Mac/Linux: source venv/bin/activate

# 4. Kutubxonalarni o'rnatish
pip install -r requirements.txt

# 5. AI modelni yuklash (bir marta)
python manage.py download_model

# 6. Migratsiyalar
python manage.py migrate

# 7. Serverni ishga tushirish
python manage.py runserver
